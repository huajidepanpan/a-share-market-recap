目录结构
--------
  scripts/
  ├── README.txt           <- 本文件（设计说明）
  ├── convert_table.py     <- Table.xls(GBK TSV) → UTF-8 CSV
  ├── market_recap.py      <- 从CSV生成分板块市场复盘报告
  └── hot_stocks.py        <- 当日热股榜获取

数据流
------
  Table.xls → convert_table.py → Table.csv → market_recap.py → recap

设计原则
--------
  1. 仅依赖 Python 标准库（无需 pip install）
  2. 每个脚本独立可运行，支持命令行参数
  3. UTF-8 编码输出，跨平台可读


一、convert_table.py
--------------------
  功能：将同花顺导出的 .xls（实际是 GBK 编码的 TSV）转为 UTF-8 CSV

  Table.xls 本质：
    - 不是真正的 Excel 文件（xlrd 无法解析）
    - 是 Tab 分隔的文本文件，编码为 GBK（或 GB2312）
    - 包含约 70 列，约 5000+ 行（全 A 股实时行情）

  处理步骤：
    a) 自动检测输入编码（GBK/GB2312/UTF-8）
    b) Tab → 逗号分隔
    c) 表头清洗（去除前置空格/点号）
    d) 输出 UTF-8 BOM 的 CSV（Excel 可直接打开）
    e) 可选：仅保留核心列（默认全量输出）

  columns_to_keep:
    默认全量输出。如需精简，核心列建议：
      序号,代码,名称,涨幅,现价,涨跌,总量,总金额,量比,换手率,
      总市值,流通市值,细分行业,5日涨幅,10日涨幅,20日涨幅,
      TTM市盈率,市净率,每股盈利,每股净资产


二、market_recap.py
-------------------
  功能：读取 Table.csv，生成分板块市场复盘报告

  板块分类与涨跌停阈值：
    ┌──────────┬──────────┬──────────┬──────────┐
    │ 板块     │ 代码特征 │ 涨停阈值 │ 跌停阈值 │
    ├──────────┼──────────┼──────────┼──────────┤
    │ 上证主板 │ SH60xxxx │  >=9.9%  │ <=-9.9%  │
    │ 深证主板 │ SZ00xxxx │  >=9.9%  │ <=-9.9%  │
    │ 创业板   │ SZ30xxxx │ >=19.9%  │ <=-19.9% │
    │ 科创板   │ SH688xxx │ >=19.9%  │ <=-19.9% │
    │ 北交所   │ 8xxxxx   │ >=29.9%  │ <=-29.9% │
    │ ST/*ST   │ *ST/ST   │  >=4.9%  │  <=-4.9% │
    └──────────┴──────────┴──────────┴──────────┘

  复盘报告内容（文本版）：
    [一] 大盘概况 —— 涨跌家数、涨跌比、成交额、平均/中位数涨幅
    [二] 分板块统计 —— 各板块涨跌家数、涨停数、成交额占比、涨幅中位数
    [三] 板块热力图 —— 动态聚合细分行业，涨停家数+成交额
    [四] 连板梯队 —— 2板以上个股按连板数分组
    [五] 分歧观察池 —— 近涨停未封+高换手，前瞻信号
    [六] 成交额 TOP20 —— 资金主攻方向
    [七] 5日涨跌 TOP15 —— 短中线趋势
    [八] 市值 TOP10 —— 大票动向
    [九] 市场总结

  HTML可视化版特性：
    - Canvas 泡泡图：X=强势家数, Y=平均涨幅, 大小=市值, 颜色=涨跌幅
    - 仪表盘计数器动画（数字从0滚动至目标值）
    - 连板梯队金字塔布局
    - 各榜单可展开/折叠（默认显示前10，点击展开全部）
    - 板块热力图下方免责备注（跨主业公司可能受第二曲线驱动）

  强势梯度定义（按板块区分）：
    主板：+7~9.9% 强势  |  +5~7% 偏强  |  +3~5% 温和
    创业板/科创板：+14~20% 强势  |  +10~14% 偏强  |  +5~10% 温和
    北交所：+20~30% 强势  |  +15~20% 偏强  |  +8~15% 温和

  使用方式：
    python market_recap.py ../20260522/Table.csv


三、hot_stocks.py
-----------------
  功能：获取当日热股榜单（东方财富人气榜 API）

  数据来源：
    - 排名：emappdata.eastmoney.com/stockrank/getAllCurrentList
    - 实时价：hq.sinajs.cn（新浪财经）

  注意：同花顺 API（eq.10jqka.com.cn）封锁外部访问，因此使用
        东方财富人气榜作为替代。两者热股榜单重叠度约 80%+。

  输出：CSV 格式，含排名、名称、代码、涨跌幅、板块、成交额

  使用方式：
    python hot_stocks.py --date 20260522

  □ 历史复盘对比

  □ 板块资金流向
    接入东方财富资金流向数据，展示板块主力净流入/流出。
